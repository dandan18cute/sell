## 技术栈说明
项目由vue-cli脚手架，通过webpack构建项目。

 1. Vue-resource和后端交互
 2. Vue-router路由实现单页应用。
 3. 第三方js库better-scroll
 4. 最大程度组件化
 5. Html5的localstorage
 6. 图标字体的使用
 7. 移动端1像素边框
 8. Css sticky footer 布局
 9. Flex弹性布局
